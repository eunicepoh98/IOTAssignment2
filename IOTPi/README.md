### Create certs folder to store your AWS Certficates
```
. 
|
└─ certs
    ├── certificate.pem.crt
    ├── private.pem.key
    ├── public.pem.key
    └── rootca.pem

```

### Create awscredentials.py file
Contents in awscredentials.py

```
accessKey = '' // AWS Access Key Id
secretAccessKey = '' // AWS Secret Access Key

```
